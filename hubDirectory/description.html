Synchronization-free chronogram of human cell cycle gene expression

We have generated a bulk RNA-seq chronogram of human cell cycle gene expression, without resorting to cell synchronization which inevitably induces distortions due to replication stress. Instead, we leveraged an RNA-preserving cell fixation method which allowed us to sort cells along the cell cycle based on DNA content (DAPI) and internal CCNA2 levels, before interrogating the resulting cell fractions using RNA-seq. We next tested for rhythmicity, i.e. assessed whether each gene displayed statistically significant oscillations in expression throughout the cell cycle, using cosinor analysis.

Due to the requirements of UCSC's bigBarChart format, in particular its incompatibility with negative values and the default log10(x+1) transformation it performs, we had to rescale the data for the purpose of display in the Genome Browser. Specifically, we min-max-scaled the expression values (Voom-normalized log2(CPM)) by the to obtain non-negative values. We then performed the following transformation: x -> (10^x)-1 in anticipation of UCSC's bigBarChart log10(x+1) default transformation. We do not advise any downstream analysis on that data, which has been optimized for display in UCSC's genome browser. Data access is described in the accompanying manuscript.

We show one track with all samples, and one track with sample averages, and report the statistical significances of the rhythmicity tests.
